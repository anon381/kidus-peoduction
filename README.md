# kidus production

This is a Next.js app that displays 'kidus production present' on the main page.




## Getting Started

1. Install dependencies:
   ```powershell
   npm install
   ```
2. Run the development server:
   ```powershell
   npm run dev
   ```
3. Open [http://localhost:3000] -     in your browser to see the result.

   ```
   (http://localhost:3000)
   ```
